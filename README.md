## Tech Stack

- Jekyll 4.3.4 with Minima theme
- GitHub Pages
- Disqus comments and Google Analytics

## Installation

```bash
git clone https://github.com/yakhyo/yakhyo.github.io.git
cd yakhyo.github.io
bundle install
```

## Development

```bash
bundle exec jekyll serve
```

Visit `http://localhost:4000`
